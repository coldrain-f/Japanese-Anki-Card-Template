## 2. 타이포그래피

### 폰트 선택 ✅ 확정

**일본어:** 유민 (Yumin) - 명조체
- 파일: `_yumin.woff` (7.6MB)
- 스타일: 우아하고 문학적인 느낌
- 웨이트: 단일 (Regular)

**한국어:** Noto Serif KR - 명조체
- 파일: `_NotoSerifKR-Regular.ttf`, `_NotoSerifKR-Medium.ttf`, `_NotoSerifKR-Bold.ttf`
- 스타일: 깔끔한 현대적 명조
- 웨이트: Regular, Medium, Bold

**선택 이유:**
- 둘 다 명조(세리프) 계열로 **통일감** 있음
- 문학적이고 고급스러운 분위기
- 일본어 학습에 어울리는 전통적 느낌

### 폰트 파일 목록

```
fonts/
├── _NotoSerifKR-Bold.ttf      (14MB)
├── _NotoSerifKR-Medium.ttf    (14MB)
├── _NotoSerifKR-Regular.ttf   (14MB)
└── _yumin.woff                (7.6MB)
```

> ⚠️ 파일명 앞 언더스코어(`_`)는 Anki 미사용 미디어 삭제 방지용

## 3. 레이아웃

### 단어 카드

```
┌─────────────────────────────────────┐
│                                     │
│           [메인 콘텐츠]              │
│                                     │
│─────────────────────────────────────│
│           [Exclude 영역]            │
└─────────────────────────────────────┘
```

**고려사항:**
- 메인 콘텐츠 중앙 정렬 vs 상단 정렬?
- Exclude 영역 구분선 필요?
- 여백(padding) 크기?

### 그룹 카드

```
┌─────────────────────────────────────┐
│                                     │
│    단어1   단어2   단어3   단어4     │
│                                     │
└─────────────────────────────────────┘
```

**고려사항:**
- 가로 나열 vs 세로 나열?
- 단어 간 간격?
- 5개 이상일 때 줄바꿈?

### 문장 카드

```
┌─────────────────────────────────────┐
│                                     │
│           [문장]                    │
│                                     │
│           [문법 태그]               │
│─────────────────────────────────────│
│           [Exclude 영역]            │
└─────────────────────────────────────┘
```

**고려사항:**
- 문법 태그 표시 스타일 (뱃지? 텍스트?)
- 긴 문장 처리 방법?

## 4. 컴포넌트 스타일

### 후리가나 (Ruby) 표시

```css
/* 옵션1: 기본 ruby */
ruby { ... }
rt { ... }

/* 옵션2: 커스텀 스타일 */
.furigana { ... }
```

**고려사항:**
- 후리가나 위치 (위/아래)
- 후리가나 색상 (메인과 동일? 연하게?)
- 글자 간격

### Exclude 영역

```
除外: 考える, 思い出す, 思いつく
```

**스타일 옵션:**
- [ ] 단순 텍스트
- [ ] 구분선 + 연한 배경
- [ ] 아이콘 + 텍스트
- [ ] 접히기(collapse) 가능

### 그룹 카드 아이템

```
┌────┐  ┌────┐  ┌────┐
│ 雪 │  │ 霧 │  │ 曇 │
└────┘  └────┘  └────┘
```

**스타일 옵션:**
- [ ] 카드형 (테두리 있음)
- [ ] 플랫 (테두리 없음)
- [ ] 그리드
- [ ] 플렉스
