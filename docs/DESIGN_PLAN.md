## 2. 타이포그래피 ✅ 확정

### 폰트 선택

**일본어:** 유민 (Yumin) - 명조체
- 파일: `_yumin.woff` (7.6MB)
- 스타일: 우아하고 문학적인 느낌

**한국어:** Noto Serif KR - 명조체
- 파일: `_NotoSerifKR-*.ttf`
- 웨이트: Regular, Medium, Bold

### 폰트 크기

| 요소 | 데스크톱 | 모바일 |
|------|----------|--------|
| 메인 단어 | 36px | 28px |
| 뜻/번역 | 22px | 22px |
| 후리가나 | 20px | 16px |
| Exclude 항목 | 13px | 14px |
| 뜻 개수 | 14px | 14px |

## 3. 레이아웃 ✅ 확정

### 단어 카드 구조

**앞면:**
```
┌─────────────────────────────────────┐
│  整える [2]                          │
│  ════════════════════════════       │
│                                     │
│  除外: [形] [姿]                     │
└─────────────────────────────────────┘
```

**뒷면:**
```
┌─────────────────────────────────────┐
│  (앞면 내용)                         │
│                                     │
│  ととのえる                          │
│  정돈하다, 조절하다                  │
│  ───────────────────────────        │
│  #1302   사전 →                      │
└─────────────────────────────────────┘
```

## 4. 컴포넌트 스타일 ✅ 확정

### 카드 헤더
- 하단에 2px 실선 (크림색)
- 메인 단어/뜻 강조

### Exclude 영역
- 인라인 스타일 (한 줄)
- 투명 배경 + 테두리만
- 라벨: "除外:"

### 사전 링크
- 심플 텍스트 링크
- 테라코타 색상
- "사전 →" 형태

### Answer 메타
- Frequency 뱃지 + 사전 링크 한 줄

## 디자인 결정 사항

| 항목 | 결정 | 이유 |
|------|------|------|
| 기본 테마 | 소프트 크림 | 눈 편안, 명조체와 어울림 |
| 일본어 폰트 | 유민 | 명조체, 문학적 느낌 |
| 한국어 폰트 | Noto Serif KR | 명조체, 유민과 통일감 |
| 강조색 | 테라코타 | 따뜻한 느낌, 크림과 조화 |
| Exclude 스타일 | 심플 인라인 | 공간 절약, 집중 방해 안 함 |
| 사전 링크 | 텍스트 링크 | 심플, 눈에 띄지 않음 |
